# Procedure Risk App

Mobile-first React/Vite app for procedure risk estimation.

## Local development

1. Install Node.js 18 or newer
2. Run:
   npm install
   npm run dev

## Deploy to Vercel

1. Create a GitHub repository
2. Upload these files
3. Sign in to Vercel
4. Import the GitHub repository
5. Click Deploy

Vercel should detect Vite automatically.

## Current features

- Procedure selection
- Bleeding modifier
- Complexity score
- Procedure tolerance score
- Frailty helper with auto-calculation
- Manual frailty override
- Quick and detailed output
- Report copy block
